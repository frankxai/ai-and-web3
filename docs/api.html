API: EVM Tool Server

Start: uvicorn agents.servers.evm_server:app --reload

GET /health

Returns { ok: true }

POST /evm/get_balance

Body: { address } → { address, wei }

POST /evm/simulate_transfer

Body: { to, value_wei } → { ok, estimated_gas|error }

POST /evm/send_transfer

Body: { to, value_wei, max_value_wei? } → tx receipt or policy error

Tools also available via the Python registry CLI: see TOOLS.md

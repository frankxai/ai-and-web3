Get Started

Follow these steps to run your first lab and build momentum.

Prerequisites

  • Python 3.10+, Node 18+, Docker
  • Foundry (or Hardhat), optional Solana CLI
  • Testnet RPC (or Anvil local)
  • Git

Setup

pip install -r requirements.txt
python scripts/check_env.py  # see missing env vars

Run Day 001

cp labs/day-001-wallets-and-rpc/.env.example labs/day-001-wallets-and-rpc/.env
python labs/day-001-wallets-and-rpc/wallet_ops.py
python scripts/generate_report.py labs/day-001-wallets-and-rpc

Tool Registry

python agents/registry.py evm.get_balance '{"address":"0x..."}'
python agents/registry.py evm.simulate_transfer '{"to":"0x...","value_wei":1000}'

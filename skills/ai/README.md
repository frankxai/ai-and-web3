# AI Skills

General AI engineering skills used across agent systems.

## Core topics
- Prompting and tool routing
- Retrieval and knowledge grounding
- Evaluation and regression tests
- Safety, policy, and spend limits
- Observability and traceability

## Useful skill slices
- eval-harness: metrics, datasets, and comparison runs
- rag-pipelines: indexing, embeddings, chunking, citations
- safety-guardrails: policy checks and approvals
- multi-agent-coordination: roles, protocols, handoffs

# Claude Code Skills

Curated skills and patterns for Claude Code workflows.

## Core skills (recommended)
- mcp-architecture: MCP server design and integration
- claude-sdk: agent SDK patterns and tool routing

## Usage notes
- Keep skills focused and runnable.
- Prefer step-based workflows with validation checks.
- Store examples as short, executable snippets.

## Where to source
- FrankX skills library (authoritative catalog): `SKILLS.md`
- Local skills (if present): `.claude-skills/`

## Contribution checklist
- Does the skill include triggers and preconditions?
- Are tools and inputs explicit?
- Is there a small test or validation step?

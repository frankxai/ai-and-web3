# Web3 Skills

Practical skills for agents that read and write on-chain.

## Core topics
- RPC and chain configuration
- Wallet ops and transaction lifecycle
- Simulation and estimation
- Indexing and data pipelines
- Storage (IPFS/Arweave)

## Skill slices
- rpc-basics: reliable providers and rate limits
- wallet-ops: balances, approvals, transfers
- tx-simulation: dry-runs and failure diagnostics
- governance: proposals and votes
- intents: planning and execution policy
